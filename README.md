## How to use it ?

You have to work on a linux computer.

### Clone the project

```bash
git clone https://github.com/Erwan-BR/System-Network-Lab-Four.git
```

### Authorize execution of bash file.

```bash
chmod +x main.sh
```

#### Execute the bash file

```bash
./main.sh
```
